1. List top five cpu hungry procesess :)

2. Get a list of all ipv4 adresses on my system

3. Count the number of subfolders a folder has
get-childitem <director> -recurse | ? {$_.psiscontainer} | measure-object | select -expand count

4. Display all .txt files in a folder including subfolders
get-childitem -recurse | where-object {$_.name -match '.*.txt$'}

5. Calculate ocupied space by all the files in a folder + subfolders
get-childitem -recurse | % {$_.length} | measure-object -sum | select -expand sum
